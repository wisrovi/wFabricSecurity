Bibliography and Source Resources
==================================

This section contains references to external resources used in the development of wFabricSecurity.

Official Documentation
---------------------

Hyperledger Fabric
~~~~~~~~~~~~~~~~~

* `Hyperledger Fabric Documentation <https://hyperledger-fabric.readthedocs.io/>`_ - Official documentation for Hyperledger Fabric
* `Fabric SDK Python <https://github.com/hyperledger/fabric-sdk-py/>`_ - Python SDK for Fabric
* `Fabric Chaincode Development <https://hyperledger-fabric.readthedocs.io/en/latest/developapps/chaincodename.html>`_ - Chaincode development guide

Python Documentation
~~~~~~~~~~~~~~~~~~~

* `Python Official Documentation <https://docs.python.org/3/>`_ - Python language documentation
* `Python Packaging Guide <https://packaging.python.org/>`_ - Package development best practices
* `PEP 8 Style Guide <https://www.python.org/dev/peps/pep-0008/>`_ - Python code style guidelines

Cryptography
~~~~~~~~~~~~

* `cryptography Library <https://cryptography.io/>`_ - Python cryptography library documentation
* `ECDSA Algorithm <https://en.wikipedia.org/wiki/Elliptic_Curve_Digital_Signature_Algorithm>`_ - Wikipedia article on ECDSA
* `SHA-2 Family <https://en.wikipedia.org/wiki/SHA-2>`_ - SHA-256 and SHA-512 algorithms

Security Standards
~~~~~~~~~~~~~~~~~

* `NIST Zero Trust Architecture <https://csrc.nist.gov/publications/detail/sp/800-207/final>`_ - NIST SP 800-207 Zero Trust Architecture
* `OWASP Top Ten <https://owasp.org/www-project-top-ten/>`_ - Web application security risks

# Ejemplo P2P - Base (Síncrono)

Envío de datos sensibles de forma privada entre Master y Slave.

## Archivos

- `master.py` - Master que envía datos sensibles
- `slave.py` - Slave que recibe datos sensibles

## Uso

```bash
# Terminal 1 - Slave
python slave.py

# Terminal 2 - Master
python master.py
```
